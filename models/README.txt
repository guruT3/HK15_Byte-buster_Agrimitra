Plant disease model (Keras)
==========================

Place your plant disease recognition model file here (or in the project root).

Accepted names:
  - plant_disease_recog_model.keras
  - plant_disease_recog_model_PwP.keras
  - plant_disease_recog_model.h5
  - plant_disease_recog_model_PwP.h5
  - any .keras or .h5 file with "plant" or "disease" in the name

The model should:
  - Input: (None, 224, 224, 3)
  - Output: (None, 39) for the 39 disease classes used by the app
